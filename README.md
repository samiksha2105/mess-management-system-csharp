 Mess Management System

 Project Description

The Mess Management System is a desktop application developed using C# and SQL to manage daily mess operations efficiently. It helps in handling student records, meal tracking, billing, and overall mess administration.

 Technologies Used

* C#
* .NET (Windows Forms / WPF)
* SQL Server

 Features
 Student Registration & Management
 Meal Tracking (Breakfast, Lunch, Dinner)
 Monthly Billing System
 Reports Generation
 Database Integration using SQL

 How to Run

1. Open the project in Visual Studio / VS Code
2. Import the database file (`.sql`) into SQL Server
3. Update database connection string
4. Run the application

 Database Setup

* Create a database in SQL Server
* Run the provided `.sql` file
* Ensure connection string is correct
